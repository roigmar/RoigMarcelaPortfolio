# Portfolio Personal - Marcela Roig

## Descripción

Este es mi **portfolio personal**, desarrollado como complemento de mi CV. Aquí presento mis proyectos, habilidades y experiencia en desarrollo web front-end, ofreciendo una visión más completa y visual de mis capacidades profesionales.

## Enlace en Vivo

Puedes visualizar el portfolio en: https://roigmar.github.io/RoigMarcelaPortfolio/
## Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos personalizados con Flexbox y Grid
- **Google Fonts** - Tipografía moderna (Poppins)
- **Diseño responsivo** - Optimizado para todos los dispositivos

## Contenido

El portfolio incluye:
- **Sobre mí** - Información profesional y personal
- **Proyectos** - Cartera de trabajos realizados
- **Habilidades** - Técnicas y competencias
- **Contacto** - Formas de comunicarse conmigo

- Diseño moderno y profesional

- Animaciones y transiciones fluidas
